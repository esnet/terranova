# Creating a Map

A map combines one or more datasets into a layered, interactive visualization. You need at least one dataset before creating a map.

## Steps

### 1. Open the map creator

In the left sidebar, click **Libraries → Maps**, then click **Create New Map**. Or navigate directly to `/map/new`.

### 2. Name your map

Enter a descriptive name in the **Name** field.

### 3. Optionally fork from an existing map

If you want to start from an existing map's configuration (viewport, layers, styling, overrides) rather than from scratch, toggle **Fork Existing Map** on and select:

- **Map Name** — the map to copy from
- **Version** — which version to use as the starting point

See [Forking a Map](forking-a-map.md) for details.

### 4. Click Create Map

Terranova creates the map and navigates to the [Map Editor](map-editor.md).

## Next steps

In the Map Editor:

1. **Add a layer** — click **Add Layer** and select a dataset
2. **Configure the layer** — set the color, edge width, and data field mappings
3. **Set the viewport** — pan and zoom to the area you want the map to show by default
4. **Save** — click **Save Map** to save your configuration

When the map is ready to share, see [Map Output](map-output.md) for embedding and export options.
